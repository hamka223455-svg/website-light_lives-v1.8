🎥 Post Reels

📌 Cara mendapatkan link foto/video:

📷 Imgur (Foto — Paling Mudah)
1. Buka imgur.com
2. Tap New Post → upload foto
3. Setelah upload, tap foto → Copy Link
4. Paste link di kolom URL
✅ Gratis, tidak perlu akun
🎬 Google Drive (Foto & Video)
1. Upload foto/video ke Google Drive
2. Tap file → titik tiga ⋮ → Share
3. Ubah ke "Anyone with the link"
4. Copy link → paste di kolom URL
✅ Gratis 15GB
🌐 Postimages (Foto)
1. Buka postimages.org
2. Upload foto → copy Direct Link
3. Paste di kolom URL
✅ Gratis, tanpa akun
⚠️ Pastikan link diakhiri dengan .jpg .png .mp4 atau link yang bisa langsung diakses publik.
